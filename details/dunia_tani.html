Dunia Tani

diketik oleh Fadli Izurohman

Category: Project

Dunia tani merupakan sebuah aplikasi web yang memfasilitasi pengguna untuk mendapatkan kompetensi dan kapabilitas untuk menghasilkan hasil tani yang lebih baik di Industri. Aplikasi ini diharapkan dapat meningkatkan edukasi pertanian sehingga meningkatkan skill para petani beserta orang-orang yang minat di bidang pertanian.

Terdapat 2 User di dalam Aplikasi Dunia Tani, yaitu Guru Tani dan User Tani. Guru tani merupakan pengguna aplikasi berbasis web “Dunia Tani” yang memproses transaksi user, memvalidasi dan memverifikasi pembayaran dan memberi akses kepada user untuk menonton video edukasi pertanian, sedangkan User Tani adalah pengguna yang melakukan transaksi seperti melakukan pembayaran courses yang terdapat pada “Dunia Tani”. Tujuan utama dari user adalah untuk melakukan transaksi Courses dan melakukan pembelajaran dunia tani.

Pada proses pembuatan Website in terdapat beberapa orang, yaitu

  • Kemal Aziz (si Project Manager)

  • Fadli Izurohman (si Back-End)

Project Dunia Tani ini merupakan sebuah tugas besar dari mata kulia Web Application Development dan dalam pengerjaannya memakan waktu selama 2 bulan. Dalam Proses pembuatannya, kami menggunakan Laravel dan bootstrap.

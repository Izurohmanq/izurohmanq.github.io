SaddleBagsCo

diketik oleh Fadli Izurohman

Category: Project

SaddleBagsCo merupakan sebuah merek dari tas motor classic. Untuk saat ini SaddleBagsco menjual 2 produk yaitu Montana dan Brownco. SaddleBagsco hadir untuk menghiasi motor classic agar terlihat lebih authentic dan makin cantik, sehingga dapat dilirik oleh masyarakat pecinta motor antik, classic, ataupun tidak.

Website SaddleBagsco in merupakan landing page. Terdapat 2 user, admin dan user biasa. Admin bisa melakukan CRUD sedangakan user biasa hanya bisa melakukan read, mengirim feedback, dan menghubungi melalui WhatsApp

Pada proses pembuatan Website ini menggunakan Laravel dan Bootstrap.
